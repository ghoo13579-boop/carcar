# 🚗 자동차 소모품 관리

차량 소모품 교체 주기를 스마트하게 관리하는 앱입니다.
**모바일 앱(Expo)** 과 **웹 버전(HTML/CSS/JS)** 두 가지 버전으로 제공됩니다.

---

## 📱 주요 기능

- **24종 차량 지원** — 현대, 기아, 제네시스, 테슬라, BMW, 메르세데스-벤츠, 토요타
- **차종별 맞춤 소모품** — 내연기관 / 전기차(EV) / 하이브리드 별도 구성
- **교체 주기 게이지** — 주행거리·날짜 중 빠른 기준으로 자동 계산
- **교체 권장 알림** — 80% 이상 도달 시 `교체 권장`, 100% 시 `교체 필요` 표시
- **사용자 주기 편집** — 각 소모품의 km/개월 주기를 직접 수정 가능
- **교체 기록 저장** — 교체 날짜·주행거리·메모를 기록, 새로고침 후에도 유지
